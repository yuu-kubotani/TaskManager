# 📂 TaskManager プロジェクト構成ガイド

このドキュメントは、TaskManager アプリケーションの「どこに・何があるか」を一目で把握するためのマップです。
新しく機能を追加したり、不具合を直したりする際の道しるべとして活用してください。

---

## 🌟 究極のフォルダ構成 (Src アーキテクチャ)

プロジェクト全体は、役割ごとに以下のフォルダに綺麗に分割されています。

| フォルダ名 | 役割・中身 |
| :--- | :--- |
| **`Src/`** | C# のプログラム本体（ソースコード）がすべて入っている最重要フォルダ。 |
| **`Data/`** | アプリが読み書きするデータ（CSVやJSON）の保管庫。 |
| **`Docs/`** | このファイルのような説明書やテンプレート類。 |
| **`Legacy_PowerShell/`** | 過去に作られたPowerShell版のスクリプト（隔離済み）。 |
| **`backup/`** | アプリが自動的に作成する日別バックアップ。 |

---

## 🛠️ 1. 開発・実行ツール (プロジェクト直下)

開発を便利にするためのスイッチ（バッチファイル）です。ダブルクリックで動きます。

* **`Run.bat`** : ▶️ アプリのビルド（翻訳）から起動までを全自動で行うメインスイッチ。
* **`TaskManager.csproj`** : ⚙️ VSCode 用の「設計図」。使う部品（ライブラリ）が書かれています。
* **`cleanup_legacy_files.bat`** : 🧹 不要なゴミファイル（obj, bin など）を一掃するお掃除スイッチ。
* **`organize_project.bat`** : 📁 ファイルが散らかった時に、この綺麗な構成へ自動整頓する魔法のスイッチ。

---

## 💻 2. プログラム本体 (`Src/` フォルダ内)

開発作業の 99% は、この `Src` フォルダの中にあるファイルを編集して行います。

### 🎨 画面・UI (`Src/Forms/`)
ユーザーが実際に操作するウィンドウ（画面）のプログラムです。
* **`FormMain.cs`** : メイン画面（リスト、カンバン、カレンダーの表示と全ての統括）。
* **`FormSettings.cs`** : 全体設定画面。
* **`FormTaskInput.cs` / `FormProjectInput.cs`** : タスク・プロジェクトの追加・編集画面。
* **`TimeLogEntryForm.cs`** : 実績時間の追加・編集画面。
* **`FormReport.cs`** : レポート分析画面。
* **`FormArchiveView.cs`** : アーカイブ画面。
* **`FormCategoryEditor.cs`** : カテゴリ編集画面。
* **`FormIcsExchange.cs`** : ICSファイル（カレンダー）の入出力画面。
* **`FormBackupRestore.cs`** : 手動バックアップ・復元画面。

### 🏗️ データ構造・設計図 (`Src/Models/`)
「データそのもの」の形（プロパティ）を定義しているファイルです。
* **`Models.cs`** : タスク、プロジェクト、予定、時間記録などのデータの「型」がまとめられています。
* **`AppSettings.cs`** : 設定画面で変更できる項目の「型」と初期値がまとめられています。

### ⚙️ 裏側の処理・データ保存 (`Src/Services/`)
画面の裏側で、データの保存や計算を行う心臓部です。
* **`DataService.cs`** : データ保存の超重要ファイル。CSVやJSONの読み書き、バックアップ作成を一手に引き受けます。
* **`CsvRepository.cs` / `JsonRepository.cs`** : 過去に使われていたデータ読み書きの部品。

### 🔧 便利な共通機能 (`Src/Utils/`)
色々な画面から共通で呼び出される、便利な「お助けツール」です。
* **`ThemeManager.cs`** / **`DarkModeRenderer.cs`** : ダークモードの適用や、色の計算を担当します。
* **`Prompt.cs`** : ちょっとした文字入力を求めるダイアログを簡単に出す機能です。
* **`NativeMethods.cs`** : Windowsの奥深い機能（マウスの放置時間の計測など）を呼び出す機能です。

## 💾 3. ユーザーデータ (`Data/` フォルダ内)

アプリを使用していく中で自動的に作成・更新されるデータ群です。
※ **絶対に手動で削除しないでください！**

* **`tasks.csv`** : 進行中の全タスクデータ。
* **`projects.json`** : プロジェクトのデータ。
* **`config.json`** : アプリの各種設定値。
* **`timelogs.json`** : 記録されたすべての作業時間（実績）データ。
* **`categories.json`** : ユーザーが作成したカテゴリ一覧。
* **`events.json`** : カレンダーの予定（イベント）データ。
* **`recurring_rules.json`** : 定期実行ルールのデータ。
* **`archived_*.csv / .json`** : アーカイブされた過去のデータ。
